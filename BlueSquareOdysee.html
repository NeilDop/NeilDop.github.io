Project Description

A simple Metroidvania style game. In this project I mainly focused on building an expandable engine, using Monogame as graphics framework. Apart from basic Jump and Run controls, I added bosses, skill relics, leveling and a save system.

Project Details

  • Scripted in C#
  • MonoGame Framework
  • Simple Physicsengine
  • Collision Detection
  • Leveling System
  • Save and Load System
  • Skill Artefacts (Wingboots for Doublejump)
  • Bulletbased Attacks

Timeframe

  • 1 Semester
  • Alongside regular University schedule
